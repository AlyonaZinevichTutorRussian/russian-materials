📚 Как пересказывать текст
How to Retell a Text
Читай шаг за шагом — и всё получится!  •  Follow the steps — you've got this! 🌟
📖
Шаг 1 Прочитай текст / Read the text
Прочитай текст внимательно.
Read the text carefully.
Если что-то непонятно — перечитай ещё раз.
If you don't understand something — read it again.
Найди незнакомые слова.
Find any words you don't know.
🧑‍🤝‍🧑
Шаг 2 О ком / о чём текст? / Who or what is the text about?
Я прочитал(а) текст о …
I read a text about …
Этот текст рассказывает о …
This text is about …
В этом тексте говорится о …
In this text we read about …
Главный герой текста — …
The main character is …
В тексте рассказывается про …
The text tells us about …
Пример / Example
🇷🇺 Я прочитала текст о маленьком лисёнке.
🇬🇧 I read a text about a little fox cub.
📍
Шаг 3 Где и когда всё происходит? / Where and when?
Действие происходит …
The story takes place …
События происходят …
The events happen …
Всё это случилось …
It all happened …
Дело было в … / Это было …
It was in … / This happened …
В то утро / В тот день …
That morning / That day …
Пример / Example
🇷🇺 Действие происходит в зимнем лесу.
🇬🇧 The story takes place in a winter forest.
▶️
Шаг 4 Что произошло сначала? / What happened first?
Сначала …
First, …
В начале текста …
At the beginning of the text, …
Всё началось с того, что …
It all started when …
Однажды … / В один из дней …
One day, … / One fine day, …
Пример / Example
🇷🇺 Сначала мальчик отправился в лес за грибами.
🇬🇧 First, the boy went into the forest to pick mushrooms.
⏭️
Шаг 5 Что было потом? / What happened next?
Потом … / После этого …
Then, … / After that, …
Затем … / Следующим был(а) …
Next, … / The next thing was …
Вдруг … / Неожиданно …
Suddenly, … / Unexpectedly, …
Тогда он(а) решил(а) …
Then he/she decided to …
В этот момент … / Тут …
At that moment, … / Just then, …
Пример / Example
🇷🇺 Потом он встретил лисёнка, который потерялся.
🇬🇧 Then he met a fox cub that had gotten lost.
🏁
Шаг 6 Чем всё закончилось? / How did it end?
В конце … / В конце текста …
In the end, … / At the end of the text, …
В итоге … / В результате …
As a result, … / Finally, …
Всё закончилось тем, что …
It all ended with …
Наконец … / Под конец …
At last, … / By the end, …
Пример / Example
🇷🇺 В конце мальчик помог лисёнку найти дорогу домой.
🇬🇧 In the end, the boy helped the fox cub find its way home.
💡
Шаг 7 Чему учит текст? / What does it teach us?
Этот текст учит нас …
This text teaches us to …
Мне кажется, этот текст о том, что …
I think this text is about …
Автор хотел показать, что …
The author wanted to show that …
Из этого текста я понял(а), что …
From this text I learned that …
Мне понравился этот текст, потому что …
I liked this text because …
Пример / Example
🇷🇺 Этот текст учит нас помогать тем, кто в беде.
🇬🇧 This text teaches us to help those who are in trouble.
